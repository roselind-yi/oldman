# 老年人社交圈子推荐系统

## 项目简介

本项目是一个面向老年人的社交圈子推荐系统，旨在帮助老年人发现符合其兴趣和地理位置的社交圈子。系统采用多源特征融合和图神经网络（GNN）技术，为用户提供个性化的圈子推荐。

## 核心功能

- **模拟数据生成**：生成包含用户、圈子和用户行为的模拟数据
- **多源特征融合**：融合文本语义、地理距离和热度特征
- **相似性搜索**：使用FAISS进行高效的相似性搜索
- **图神经网络**：构建用户-圈子交互图并训练GNN模型提升推荐效果

## 项目结构

```
oldman/
├── data/              # 数据存储目录
├── data_generation/   # 模拟数据生成模块
├── feature_engineering/ # 特征工程模块
├── models/            # 模型模块
├── config.py          # 配置文件
└── main.py            # 主入口文件
```

## 技术栈

- Python 3.9+
- pandas、numpy：数据处理
- sentence-transformers：文本嵌入
- FAISS：相似性搜索
- PyTorch、PyTorch Geometric：GNN模型
- geohash2：地理编码处理

## 快速开始

### 环境配置

1. 克隆项目到本地

2. 安装依赖：
   ```bash
   pip install pandas numpy sentence-transformers faiss-cpu torch torch-geometric geohash2 faker
   ```

### 运行项目

```bash
python main.py
```

运行流程：
1. 检查并生成模拟数据（如果不存在）
2. 提取并融合多源特征
3. 演示基于FAISS的相似性搜索
4. 构建图数据并训练GNN模型

## 模块说明

### 1. 数据生成模块 (`data_generation/`)

- `mock_data_generator.py`：生成模拟的用户数据、圈子数据和用户行为数据

### 2. 特征工程模块 (`feature_engineering/`)

- `feature_extractor.py`：提取文本特征并生成嵌入向量
- `multi_feature_fusion.py`：融合文本、地理和热度特征

### 3. 模型模块 (`models/`)

- `faiss_indexer.py`：使用FAISS构建索引并进行相似性搜索
- `graph_builder.py`：构建用户-圈子交互图
- `gnn_model.py`：定义和训练GNN模型

### 4. 配置文件 (`config.py`)

包含项目的各种配置参数，如数据路径、模型参数等。

## 推荐流程

1. **特征提取**：将用户兴趣和圈子信息转换为文本嵌入向量
2. **特征融合**：融合文本、地理和热度特征
3. **初步搜索**：使用FAISS进行相似性搜索获取候选圈子
4. **重排序**：基于多源特征融合进行重排序
5. **GNN优化**：使用GNN模型学习用户-圈子交互模式，进一步优化推荐

## 配置说明

在 `config.py` 文件中，您可以调整以下配置：

- `NUM_USERS`：生成的用户数量
- `NUM_CIRCLES`：生成的圈子数量
- `MAX_BEHAVIORS_PER_USER`：每个用户的最大行为记录数
- `INTEREST_TAGS`：兴趣标签池
- `GEO_RANGE`：地理坐标范围
- `SENTENCE_MODEL_NAME`：使用的预训练模型
- `FEATURE_WEIGHTS`：特征融合权重
- `GNN_HIDDEN_CHANNELS`：GNN隐藏层通道数
- `GNN_LEARNING_RATE`：GNN学习率
- `GNN_EPOCHS`：GNN训练轮数
